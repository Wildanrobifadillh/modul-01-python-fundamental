# Modul 01 - Python Fundamental + Secure Input

Repository ini berisi latihan saya di Modul 01, yaitu dasar-dasar Python dari awal. Fokus utama modul ini bukan cuma membuat program bisa jalan, tapi juga mulai membiasakan cara menulis kode yang rapi, mudah dibaca, dan tidak mudah rusak saat menerima input dari user.

Di modul ini saya belajar tentang variabel, tipe data, input, output, operator, percabangan, dan validasi input dasar.

## Tujuan Modul

Tujuan dari modul ini adalah membangun dasar Python yang kuat sebelum masuk ke materi yang lebih besar seperti function, list, dictionary, project kalkulator, dan program kasir sederhana.

Hal yang saya latih di modul ini:

- Membuat variabel dan menyimpan data
- Memahami tipe data seperti string, integer, float, dan boolean
- Menampilkan output dengan `print()`
- Membuat output lebih rapi dengan f-string
- Menerima input dari user dengan `input()`
- Membersihkan input dengan `strip()`
- Mengubah input angka dari string ke integer dengan `int()`
- Mengecek input angka dengan `isdigit()`
- Menggunakan operator aritmatika dan perbandingan
- Membuat percabangan dengan `if`, `elif`, dan `else`
- Membuat validasi input agar program tidak mudah error
